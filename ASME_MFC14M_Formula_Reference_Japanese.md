## 主要な計算式

### 放出係数（Discharge Coefficient）

#### SI単位での基本式

$$C = \frac{q_m}{\frac{\pi}{4}d^2\sqrt{\frac{2\Delta p \rho_f}{1-\beta^4}}} \quad (3-1a)$$

#### 慣用単位での基本式

$$C = \frac{q_m}{0.09970190 d^2\sqrt{\frac{h_w - \rho_f}{1-\beta^4}}} \quad (3-1b)$$

### 質量流量（Mass Rate of Flow）

#### SI単位

上流測定：
$$q_m = \frac{\pi}{4}C\varepsilon_1 d^2\sqrt{\frac{2\Delta p\rho_f}{1-\beta^4}} \quad (3-11a)$$

下流測定：
$$q_m = \frac{\pi}{4}C\varepsilon_2 d^2\sqrt{\frac{2\Delta p\rho_2}{1-\beta^4}} \quad (3-12a)$$

#### 慣用単位

上流測定：
$$q_m = 0.09970190 C Y_1 d^2\sqrt{\frac{h_w\rho_f}{1-\beta^4}} \quad (3-11b)$$

下流測定：
$$q_m = 0.09970190 C Y_2 d^2\sqrt{\frac{h_w\rho_2}{1-\beta^4}} \quad (3-12b)$$

### レイノルズ数（Reynolds Number）

#### SI単位

$$R_D = \frac{U_f D}{\nu_f} = \frac{4q_m}{\pi \mu_f D} \quad (3-13a)$$

#### 慣用単位

$$R_D = \frac{U_f D}{V} = \frac{227738q_m}{\mu_f D} \quad (3-13b)$$

径比に基づく場合：
$$R_d = \frac{R_D}{\beta} \quad (3-14)$$

### 熱膨張補正（Thermal Expansion Correction）

流動条件での直径 d_f と D_f：

$$d_f = [1 + \alpha_{pe}(t_f - t_{meas})]d_{meas} \quad (4-3)$$

$$D_f = [1 + \alpha_p(t_f - t_{meas})]D_{meas} \quad (4-4)$$

### 等エントロピー指数（Isentropic Exponent）

等エントロピー指数と比熱の関係：

$$\kappa = \frac{\rho}{\rho}\left|\frac{\partial p}{\partial \rho}\right|_s = \frac{p^2}{p} \frac{C_p}{C_v} p\left|\frac{\partial p}{\partial \rho}\right|_T = \frac{c^2}{p} \quad (3-8)$$

### ガス（蒸気）密度（Gas/Vapor Density）

#### SI単位

$$\rho_f = 0.003483407 \frac{p_f G_f}{Z_f T_f} \quad (4-6a)$$

#### 慣用単位

$$\rho_f = 2.698825 \frac{p_f G_f}{Z_f T_f} \quad (4-6b)$$

### 膨張係数（Expansion Factor）

上流測定に基づく膨張係数：

#### SI単位
$$\varepsilon_1 = 1 - \frac{(0.41 + 0.35\beta^4)\Delta p}{\kappa p_{1}} \quad (7-3a)$$

#### 慣用単位
$$Y_1 = 1 - \frac{[0.41 + 0.35\beta^4]h_w}{27.73\kappa p_1} \quad (7-3b)$$

下流測定に基づく膨張係数：

#### SI単位
$$\varepsilon_2 = \sqrt{1 + \frac{\Delta p}{p_{12}}} \cdot \varepsilon_1\sqrt{1 + \frac{\Delta p}{p_{12}}} \quad (3-5a)$$

#### 慣用単位
$$Y_2 = Y_1\sqrt{1 + \frac{27.73h_w}{p_{12}}} \quad (3-5b)$$

### コーナータップ用放出係数（Corner Tap Configuration）

$$C = \left[0.5991 + \frac{0.0044}{D} + \left(0.3155 + \frac{0.0175}{D}\right)(\beta^4 + 2\beta^8)\right]\sqrt{1-\beta^4}$$
$$+ \left[\frac{0.52}{D} - 0.192 + \left(16.48 - \frac{1.16}{D}\right)(\beta^4 + 4\beta^8)\right]\frac{\sqrt{1-\beta^4}}{\sqrt{R_D}}$$

ここで：
- D = メータチューブ内径 (in.)
- R_D = パイプレイノルズ数
- β = オリフィス径/パイプ径

### フランジタップ用放出係数（Flange Tap Configuration）

$$C = [0.5980 + 0.468(\beta^4 + 10\beta^2)]\sqrt{1-\beta^4}$$
$$+ (0.87 + 8.1\beta^4)\frac{\sqrt{1-\beta^4}}{\sqrt{R_D}}$$

ここで：
- D = メータチューブ内径 (in.)
- R_D = パイプレイノルズ数
- β = オリフィス径/パイプ径

### 圧力損失（Pressure Loss）

#### SI単位

$$\Delta\omega = \frac{1-\beta^4 - C\beta^2}{1-\beta^4 + C\beta^2}\Delta p \quad (9-1a)$$

#### 慣用単位

$$h = \frac{1-\beta^4 - C\beta^2}{\sqrt{1-\beta^4 + C\beta^2}}h_w \quad (9-1b)$$

### 不確かさ計算（Uncertainty Calculation）

質量流量の相対不確かさ：

#### SI単位

$$e_r = \frac{\delta q_m}{q_m} = \sqrt{\left(\frac{\delta C}{C}\right)^2 + \left(\frac{\delta \varepsilon}{\varepsilon}\right)^2 + \left(\frac{2\beta^4 \delta D}{1-\beta^4 D}\right)^2 + \left(\frac{2\delta d}{1-\beta^4 d}\right)^2 + \left(\frac{\Delta(\rho_1)}{2\rho_1}\right)^2 + \left(\frac{\delta \rho}{2\rho}\right)^2}$$

#### 慣用単位

$$e_r = \frac{\delta q_m}{q_m} = \sqrt{\left(\frac{\delta C}{C}\right)^2 + \left(\frac{\delta Y}{Y}\right)^2 + \left(\frac{2\beta^4 \delta D}{1-\beta^4 D}\right)^2 + \left(\frac{2\delta d}{1-\beta^4 d}\right)^2 + \left(\frac{\delta h_w}{2h_w}\right)^2 + \left(\frac{\delta \rho_1}{2\rho_1}\right)^2 + \left(\frac{\delta \rho}{2\rho}\right)^2}$$

## 計算例

### 例1：質量流量の計算（コーナータップ構成）

**与えられた条件：**
- パイプ内径 D = 25 mm = 0.984 in.
- オリフィス直径 d = 15 mm = 0.591 in.
- 差圧 Δp = 5000 Pa
- 流体密度 ρ_f = 1000 kg/m³
- 放出係数 C = 0.62
- 膨張係数 ε₁ = 1.0（液体）

**径比の計算：**
$$\beta = \frac{d}{D} = \frac{15}{25} = 0.6$$

**質量流量（SI単位）：**
$$q_m = \frac{\pi}{4} \times 0.62 \times 1.0 \times (0.015)^2 \times \sqrt{\frac{2 \times 5000 \times 1000}{1-0.6^4}}$$

$$q_m = \frac{\pi}{4} \times 0.62 \times 1.0 \times 0.000225 \times \sqrt{\frac{10,000,000}{1-0.1296}}$$

$$q_m = 0.0001198 \times \sqrt{\frac{10,000,000}{0.8704}} = 0.0001198 \times 3395.3 = 0.407 \text{ kg/s}$$

### 例2：レイノルズ数の計算

**与えられた条件：**
- 質量流量 q_m = 0.407 kg/s
- 流体粘度 μ = 0.001 Pa·s
- パイプ直径 D = 0.025 m
- 流体密度 ρ = 1000 kg/m³

**レイノルズ数（SI単位）：**
$$R_D = \frac{4 q_m}{\pi \mu D} = \frac{4 \times 0.407}{\pi \times 0.001 \times 0.025} = \frac{1.628}{0.0000785} = 20,740$$

これは推奨値 R_D > 1000 を満たしている。

## 不確かさの源と推定値

| 不確かさの源 | 典型的な推定値 |
|-----------|------------|
| 放出係数 (C) | ±0.75% |
| 膨張係数 (ε) | ±0.144% h_w/p₁ |
| パイプ直径測定 (D) | ±0.4% |
| オリフィス直径測定 (d) | ±0.07% |
| 差圧測定 (Δp) | ±0.5% ～ ±2.0% |
| 密度測定 (ρ) | ±0.5% ～ ±2.0% |
| 直管長が短い場合 | ±0.3% |
| **合計（二次結合）** | **±1.0% ～ ±2.5%** |
